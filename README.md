# Logic Gate Stencil

A 3D printable stencil for drawing electronic circuit symbols easily.

## Features
- AND gate
- OR gate
- XOR gate
- Triangle
- Capacitor
- Resistor holes

## Purpose
This stencil helps students quickly draw circuit diagrams on paper.

## Files
Stunning Lappi.stl – 3D printable stencil

## How to Use
1. Download the STL file
2. Print using a 3D printer
3. Use a pencil to trace logic gate shapes
